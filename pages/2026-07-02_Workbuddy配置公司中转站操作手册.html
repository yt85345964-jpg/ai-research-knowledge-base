Workbuddy 配置公司中转站操作手册

来源：IMA「杨添的知识库」→ PDF

核心内容

该文档记录了将 WorkBuddy 接入公司内部 AI 网关（华诺星空 AI 网关）的完整配置流程：

  1. **获取 API 令牌**：通过公司 OA 系统的"AI 网关"入口 → 令牌管理 → 添加令牌，选择模型 deepseek-v4-pro，复制密钥
  2. **配置 WorkBuddy 自定义模型**：设置 → 模型 → 新增模型 → 自定义，填入：
  3. - 接口地址：`http://aigateway.novasky.cn/v1`（兼容 OpenAI 协议）
  4. - API Key：从网关复制的令牌
  5. - 模型名称：deepseek-v4-pro
  6. - 上下文窗口：输入 1M / 输出 384K
  7. **切换使用**：在对话界面底部将 auto 切换为自定义模型

关键洞察

  1. **WorkBuddy 支持 OpenAI 兼容协议的自定义模型接入**，不限于内置模型商，可通过 API 网关接入企业私有部署或中转的模型
  2. **企业 AI 网关模式**：公司提供统一的令牌管理和流量控制，员工只需复制密钥配置即可使用，降低了每人的接入门槛
  3. **配置参数透明**：接口地址、API Key、模型名称、上下文窗口四个参数即可完成接入，说明 WorkBuddy 的自定义模型功能设计面向通用兼容性
