核心内容（基于 ASR 转录全文提炼）

作者（第四种黑猩猩）2026上半年亲自做了大量企业 AI 落地（自主培训 + FDE 前线部署工程师双线），视频提出三个思考：

思考一：市场行情

  • - **两个判断**：①企业 AI 服务缺口极大，市场才刚开始；②市场虽大，但选对客户极关键。
  • - **现实落差**：99% 的公司还没真正落地，多数停留在「问豆包/大模型问题」阶段；技术端与企业端之间存在巨大「信息鸿沟」，企业需要既懂技术又懂商业/产品/运营的综合性人才。
  • - **选客户三标准**：①原业务尽量数字化（少线下环节）；②从业人员学习/适应能力强；③**有钱**（能接受新事物、付得起服务溢价）。

思考二：人的因素（两个反直觉）

  • - **老板反直觉**：看后台 token 消耗，企业里用 Agent 最猛的恰恰是部门负责人/一把手——「token 不会说谎」。老板想法多但过去执行慢，Agent 相当于随叫随到的执行者，想法可快速验证。**结论：一把手自己不上手亲自用 Agent，企业 AI 落地基本推不动。**
  • - **员工反直觉**：目标不该是「提升整个部门效率」，而是「用 AI 扩展每个员工个人能力」（把部门做成 10 个超级个体）。原因：①当前 Agent 设计（受 Claude/某 Peter 影响）本就是为个人服务；②公司用熟 AI 后最慢环节是人，工作关系分人-人 / 人-Agent / Agent-Agent，最慢的是人-人，故应扩大个人能力边界、减少无意义人际接触。

思考三：具体怎么做 —— 公式 agents + skills + data（智能体+技能+数据）

  • - **agent（最不重要）**：直接用世界上最好的（Codex、Claude Code 等），别自己折腾——你不是 Agent 创业公司，折腾不过别人。
  • - **skills（技能）**：把所有重复性工作（流程手册、甚至重复认知）做成 skills 供 Agent 调用。两阶段：①把原流程做成 skills（多数企业卡在这）；②**按 AI 时代特点把流程重做一遍**（真正提升）——类比电脑刚出现被当打字机、打印归档；今天 Agent 生成方案却丢回老 SaaS 系统，流程改造是难点，但优化好流程即成「公司第二大数字资产」。
  • - **data（数据，最难、第一大数字资产）**：两层难点——①怎么搞到数据（线上会议自动记录、线下配硬件、官方接口/爬虫）；②怎么组织管理（小范围用 Wiki 知识库、何时用传统 DB、何时向量 DB、每日新增如何管理）。**胜负手**：Agent 与模型是公开花钱能买的，差距在「谁的数据好、管理得好」。
  • - **总结**：企业 AI 落地的胜负手从不在最酷炫的 Agent，而在最不起眼处——先换习惯 → 学用 Agent → 重做流程 → 把数据一点点喂进去，过程又慢又脏，但正是它构成企业自己的护城河。

关键洞察

  1. **一人公司/个体户企业AI服务模式正在兴起**：作者"一个人亲手做企业AI服务"，印证了 AI使用哲学 中 Builder 心态的现实路径——个人也可以交付企业级 AI 服务。
  2. **视频多平台分发验证**：同一内容在B站、抖音、头条均有分发，作者（第四种黑猩猩CHIMP）在多个平台有账号，说明个体 AI 从业者正在形成跨平台内容IP。
