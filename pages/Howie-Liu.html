Howie Liu

核心定义

**一句话定位**：Airtable 联合创始人兼 CEO，HyperAgent 创建者；以「Builder 心态 + 判断力规模化」与「把 Agent 当员工来雇」的论断，成为 AI 原生工作方式的代表性产品思考者。

主要贡献

  • - Airtable 联合创始人兼 CEO（2013 年创立），估值约 117 亿美元；个人净资产估算 10–12 亿美元（主要源于 Airtable 股权）。
  • - 13 岁自学 C++；杜克大学机械工程与公共政策专业（2005–2009），在校结识日后共同创立 Airtable 的 Andrew Ofstad 与 Emmett Nicholas。
  • - 第一家公司 Etacts（CRM，YC 孵化，2011 年被 Salesforce 收购）；之后创立 Airtable，主打「无代码 + 关系型数据库」让非技术团队也能搭应用。
  • - 现推动 Airtable 向 AI 原生演进，推出独立多智能体产品 Superagent（2026 年 1 月）与面向 Agent 创业者的 HyperAgent「Founding 500」计划（投入 1000 万美元推理额度给 500 位创始人）。
  • - 在《Silicon Valley Girl》播客中提出核心框架：AI 时代拉开差距的不是工具使用能力，而是**发现并定义问题的能力（Problem Hunting）**与**把好判断力规模化输出的能力（判断力飞轮）**——即 Builder 心态 + 判断力规模化。该访谈是 AI使用哲学 页的核心来源。
  • - 2026 年 AI Engineer World's Fair 提出「雇一个 Agent 当员工」：画了 Agent 产品演进五级阶梯——① completions → ② chatbot → ③ agent（自主调工具/多步推理）→ ④「claw」（连续工作数小时、靠心跳自唤醒，借用 OpenClaw 一词）→ ⑤ agent 对 agent 编排。
  • - 提醒：早年 BabyAGI、AutoGPT 等「自主 agent」虽能连跑数小时但易跑偏；现模型能在更长开放任务上保持方向感，才是「claw」级落地的真正前提。
  • - 用真实客户（一家园艺翻新线下店）演示「一支 agent 员工团队」跑通从获客到成交的全链路：分诊 agent → 勘测 agent → … 编排协作。

代表观点

  • - 「能把好判断和精准反馈高效地施加到尽可能多的线程上的人，会变成几乎是超人。」
  • - 「找到问题占了整个事情的 80%。」（Problem Hunting）
  • - 「Agent 不是来让你把现有的事做到极致的，是让你去做那些你本来做不了的事。」
  • - 「奢侈雇员」（Luxury Hires）：Agent 让旅行规划师、行情追踪等「不值得专门雇人」的岗位变得可行，24 小时运行、无需工资。
