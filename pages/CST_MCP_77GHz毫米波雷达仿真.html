CST Runtime CLI 是什么？

仓库：github.com/bbl21/cst-runtime-cli（MCP Server）

能力：让 AI Agent 通过 MCP 协议调用 CST Studio Suite 完成电磁仿真任务。

安装/使用：


# 初始化（MCP 服务 + 优化 skill）
# 详细见 skills/cst-runtime-cli/references/setup_guide.md
# 关键：所有命令必须使用 uv run 模式，不可走系统 Python

# Agent 使用方式：告诉它需求
# 源工程: C:\path\to\your_project.cst
# 优化目标: 例如 9-11 GHz S11 ≤ -40 dB
# 可调参数: 例如 g (20-30, 步进0.5)
# Agent 会先读取工程参数列表让用户确认，然后自动执行

77GHz 汽车毫米波雷达仿真要点

CST 在毫米波雷达开发中的关键应用

  1. **天线阵列设计与优化**——支持 77GHz/79GHz 频段，精准优化波束成形和辐射特性
  2. **射频前端系统集成**——电路-电磁协同仿真，验证功率放大器/混频器等模块匹配性能
  3. **环境场景仿真验证**——雷达回波仿真，多目标、多径效应
  4. **EMC/EMI 性能预测**——设计阶段预判电磁兼容问题

关键参数

参数 典型值 物理意义
中心频率 78.5GHz 载波频率
信号带宽 4GHz 决定距离分辨率
Chirp 时长 50μs 单个调频周期
采样间隔 0.1ns ADC 采样时间

距离分辨率公式


ΔR = c / (2B)
# c = 光速
# B = 信号带宽

求解器选择

求解器 适用场景
时域（T-solver） 宽带分析、非线性器件
频域（F-solver） 高 Q 值谐振结构
积分方程法 辐射问题、开放边界
多层快速多极子 超电大尺寸问题
TLM 全波 高精度（资源密集）
A-solver NFS + Radome 组合仿真

雷达罩建模技巧（77GHz 保险杠）

  • - 多层薄壁结构用 `Thinpanel` 建模（避免网格爆炸）
  • - 每层定义材料参数 → 设置层叠列表 → 3D 薄层等效
  • - WCS 局部坐标系指示层叠方向

双求解器交叉验证

一定要用不同的解算器来做对比——如果不同算法都能拿到相同的结果，才能说明模拟是准确的。
  • - Boundary reflection level ≤ 1e-6
  • - F 求解器：精度 1e-6 起步，勾选网格自适应加密
  • - T 求解器：boundary 与 F 相同，重点在 local mesh 加密

关键洞察

  1. **AI Agent + 仿真软件 = 工程师的"AI 副驾"**：自然语言任务 → 自动建模 → 自动仿真 → 自动导出 → 自动对比——传统需要数天的仿真优化工作可压缩到数小时。
