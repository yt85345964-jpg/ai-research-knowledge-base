AI 辅助硬件研发工作流

核心定义

**一句话定位**：Claude（Code）已可深度介入软硬件研发全流程——从嘉立创 EDA 自动画原理图、到嵌入式 MCU 远程调试（AI 调试器 AixProbe），再延伸到平衡车/自动驾驶等具体产品形态，AI 正在成为"看得见硬件、动得了硬件"的研发副驾。

目标

让 AI 深度介入软硬件研发全流程：从 EDA 自动画原理图/PCB，到嵌入式 MCU 远程调试（读寄存器、烧录、单步），再到具体产品形态落地；缩短设计周期、保证一致性、沉淀可复用的设计经验。

适用场景

  • - 原理图/PCB 自动生成（嘉立创 EDA + Claude Code，已成熟）
  • - 嵌入式远程调试（JTAG/SWD + MCP，AixProbe）
  • - 仿真自动化（CST Runtime CLI 经 MCP）
  • - 平衡车/自动驾驶等具体产品实战跳板
  • - 边界：AI 硬件操作依赖网关设备与 MCP 接口；仿真自动化处于接入扩展期

流程阶段 · 角色 · AI能力

阶段 负责角色 AI 能力 输入 → 输出 工具

关键步骤

  1. **需求与设计（自动画原理图）**：安装 easyeda-api skill（`claude mcp add --transport stdio kicad ...` 或 `https://github.com/easyeda/easyeda-api-skill`）；嘉立创 EDA 侧高级 → 扩展管理器 → 搜 "run" → 安装 → 勾选三个选项；`claude code` 中输入 "EDA 启动！" 建立 WebSocket 连接。价值：减少重复手动操作、一致性保证、知识沉淀为自动化脚本、工程师专注创造性设计。
  2. **调试（AI 远程调试器 AixProbe）**：基于 T113-S3 微型网关 + 内置 OpenOCD + WiFi 6 + JTAG/SWD + MCP 服务接口；CLI 与 GUI 两种配置；上位机自动切换 ssh/adb，电脑端双击单文件 exe 运行。AI 从"只看代码"升级为直接操作硬件（读寄存器、烧录、单步）。
  3. **仿真验证（AI 仿真自动化）**：通过 CST Runtime CLI 经 MCP 将仿真软件 AI 化，模型 → S 参数 / 远场。
  4. **产品化（平衡车/自动驾驶跳板）**：平衡车涵盖电源管理（锂电池+DC-DC+LDO）+ MCU（GD32F303C8T6）+ 运动传感器 + 电机驱动（TB6612）+ 蓝牙 + 编码器/红外，几乎覆盖所有嵌入式核心模块；AI 参与电源/信号完整性、EMC 早期评估，自动检测未连接引脚、相似器件推荐、连接关系检查。

效果指标

  • - 设计周期缩短：减少重复性手动操作，自动化画板/调试压缩研发时间。
  • - 一致性提升：自动化避免人为错误，保证设计一致性。
  • - 知识沉淀：设计经验 → 可复用自动化脚本（Skill / CLAUDE.md）。

异常处理

  • - 嵌入式调试盲区（过去 AI 只能看代码、猜硬件 Bug）→ 用 AixProbe 通过 JTAG/SWD + MCP 直接操作硬件读寄存器、烧录、单步。
  • - 网关/连接异常 → 上位机自动切换 ssh/adb；电脑端双击单文件 exe 运行。
  • - 仿真软件未接入 → 通过 CST Runtime CLI 经 MCP 接入（CST、HFSS、ADS 等后续都将 MCP 化）。
  • - 硬件可靠性风险（动态系统毛刺"翻车"）→ 在平衡车等高要求场景用 AI 早期评估电源/信号完整性、EMC。

实施方法与效果

  • - 实施方法：
  • - 嘉立创 EDA + Claude Code：安装 easyeda-api Skill → EDA 侧装 Run 扩展 → 连接建立 WebSocket → 自然语言驱动画板。
  • - AixProbe：部署 T113-S3 网关 + OpenOCD + MCP，CLI/GUI 配置，接入 AI 远程调试。
  • - 工具栈：Claude Code（AI 大脑）、easyeda-api（工作流引擎）、Run API Gateway（EDA 侧暴露 99% 内部 API）、EasyEDA Agent Connector（本地 daemon ↔ eda.* API）、KiCad MCP、华秋 KiCad Copilot、NextBoard（硬件方案设计 Agent）、AixProbe、atopile、CST Runtime CLI。
  • - 效果评估：
  • - AI 硬件研发范式已成型：画板 → 调试 → 产品全链条都有 AI 切入点。
  • - "AI 看硬件、动硬件"是 2026 关键突破。
  • - 国产 EDA 主动拥抱 AI（嘉立创 EDA 推 Run API Gateway、easyeda-api Skill、Agent Connector 三套方案）。
  • - 经验总结：
    • 平衡车是 AI 嵌入式练手神器（挑战性、模块丰富度、社区资源兼具），是验证 AI 全栈辅助的最佳载体；AI 仿真自动化是研发副驾的下一站（CST、HFSS、ADS 等经 MCP 接入 Agent）。
