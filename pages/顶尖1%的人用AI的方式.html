顶尖1%的人用AI的方式，跟你完全不一样

来源: 微信公众号「是大鱼呀」，2026-06-27 发布

主题: Airtable CEO Howie Liu 在《Silicon Valley Girl》播客中的访谈解读

核心人物: Howie Liu（Airtable CEO，Hyper Agent 创建者）

一句话总结

AI 时代真正拉开差距的不是工具使用能力，而是发现并定义问题的能力和把好判断力规模化输出的能力——即 Builder 心态 + 判断力飞轮。

关键观点

1. Builder 心态（Tinker）

  • - "tinker"是核心词：折腾、捣鼓、自己上手试试
  • - 没人知道"正确答案"，最有效的用法是 Builder 社区里的人"不小心"发现的
  • - Builder ≠ 会写代码，而是愿意下场动手、驱动工具而不是等工具告诉你能做什么
  • - 类比个人电脑诞生：让算力变简单到任何人都能用，而不是做更晦涩更专业的项目

2. 判断力是唯一不会被压缩的竞争力

  • - AI 让执行成本趋近于零，你从"做事的人"变成"做决策的人"
  • - "能把好判断和精准反馈高效地施加到尽可能多的线程上的人，会变得几乎是超人"
  • - 判断力像肌肉：做的判断越多，下一次越准。Agent 提供前所未有的训练场：一天从 3 个决策到 30 个
  • - Agent 替代执行层，判断层（品味、人性理解、模糊信号取舍）永远是人的领地

3. Problem Hunting 占了 80%

  • - 过去：找问题容易，构建解决方案难（要写代码、雇人、花钱）
  • - 现在：比例反转。构建方案已不难（Agent 去执行），真正的难度全在：**你到底想解决什么问题？**
  • - 建议：先列清单，把反复在做/反复在烦/一直缺失的事全列出来，不要上来就想"我要搭什么 Agent"
  • - 问题清单 = Agent 部署计划

4. 奢侈雇员（Luxury Hires）

  • - 以前太多事不值得专门雇人：旅行规划师、二手车行情追踪……
  • - Agent 让这些"奢侈岗位"变得可行：设定偏好和规则，24 小时跑，不需要工资
  • - 商业层面：同时跑 10 个营销实验、5 个产品方向、20 种客户关系
  • - "更多的击球机会" → 犯更多错、验证更多假设、找到更多正确方向
  • - 飞轮：Agent 跑更多实验 → 判断力积累更快 → 设计更好方向 → Agent 跑更多实验

5. 入门的正确顺序

  1. 找一个好用的 Agent 工具上手，不要纠结选哪个
  2. 用低风险个人场景入手，不要上来就改造业务流程
  3. 把它当乐趣，不是任务。先作为消费者沉进去，摸清逻辑再上商业任务
