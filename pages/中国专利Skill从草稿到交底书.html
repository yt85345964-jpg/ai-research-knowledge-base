一句话总结

patent-disclosure-skill 是面向 Claude Code / Cursor 的 AI Agent Skill，通过多轮自适应对话引导发明人从零梳理技术方案，自动生成符合中国国家知识产权局规范的完整专利交底书，支持 SVG 附图自动生成和 Word 一键导出。

两个版本

版本 仓库 特点
fuyuxiang 版 github.com/fuyuxiang/patent-disclosure-skill 多轮自适应对话 + 三大技术领域覆盖 + SVG 附图 + Word 导出
Sven-LI-sankyuu 版 github.com/handsomestWei/patent-disclosure-skill 专利点挖掘 + 联网国知局查新 + 脱敏成文 + 自检闭环

fuyuxiang 版核心能力

  1. **多轮自适应对话**：根据用户角色（发明人/代理人）和输入材料（口头描述/文档/代码）自动调整对话深度
  2. **三大技术领域覆盖**：内置计算机软件、机械制造、电子通信领域术语表和专业句式
  3. **专利语言规范化**：自动应用专利文书惯用表达，过滤 AI 高频套话
  4. **SVG 附图自动生成**：系统架构图、方法流程图、数据流图，严格遵循专利附图要求（纯黑白、三位数字标记、引线规范）
  5. **Word 一键导出**：A4 排版 .docx（宋体小四、1.5 倍行距、首行缩进）
  6. **分章节确认**：核心章节逐个生成、逐个确认，避免大段返工

Sven-LI-sankyuu 版核心流程

  1. **项目扫描与专利点挖掘**：扫描代码/文档，先列候选专利点再筛选
  2. **国知局查新**：优先走 cnipa.gov.cn，四层降级（Playwright → requests → curl → Google Patents），相关性判断准确率 95%+
  3. **脱敏成文**：自动过滤敏感信息，生成符合规范交底书
  4. **自检闭环**：内置校验规则确保完整性

关键洞察

  1. **专利撰写是研发部门的通用需求**：无论硬件AI/嵌入式AI/算法和数据处理AI，都可能需要专利交底书。这个 Skill 对**跨≥2研发部门**有价值，归入研究公共AI。
  2. **"查新→成稿→自检"闭环是关键创新**：不是简单让 AI 写一篇交底书，而是先查国知局现有专利判断新颖性，再生成，再自检——这个闭环比纯生成靠谱得多。
  3. **Skill 正在从"通用助手"向"专业领域工具"演进**：专利撰写 Skill、CST 仿真 Skill、EDA Skill——每个领域都在涌现专业 Agent 工具。
  4. **对公司的直接价值**：公司处于上市辅导期，专利是重要资产。这个 Skill 可帮助团队从技术方案快速生成交底书初稿，节省代理机构前期沟通时间。
